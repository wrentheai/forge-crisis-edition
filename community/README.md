# Community

## Discord Bot

Run Forge as a Discord bot so your whole server can query it.

### Setup (coming soon)
```bash
pip install discord.py ollama
python bot.py
```

The bot listens for `!forge <question>` and returns Forge's response.

### Commands
- `!forge <question>` -- ask Forge anything
- `!forge-challenge` -- get this week's Junk Drawer Challenge
- `!forge-leaderboard` -- see top savers this month

## Weekly Junk Drawer Challenge

Every Monday, a new challenge drops:

> **Fix or build something using ONLY what's in your house right now. No buying anything. Post your result.**

Past challenges:
- Week 1: Build a phone stand from cardboard
- Week 2: Fix a leaky faucet without new parts
- Week 3: Make a solar phone charger from old electronics

## Sharing Your Saves

Tag your fixes with **#ForgeSaved** and the dollar amount. We track community savings.
